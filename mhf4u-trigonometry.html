4. Functions

f(x) = asin(kx - p) + d

a = Vertical stretch. This number is also the amplitude of the function. If a < 0, then the graph is reflected on the x-axis.
k = Horizontal stretch/compression. You use k to find the period of a function, which is calculated by 360/k if in degrees or 2πk in radians. If k < 0, then the graph is reflected on the y-axis.
p = Horizontal translation. Also called phase shift.
d = Vertical translation. To determine the horizontal axis (centre) of the wave, do:
lowest point + highest point / 2

f(x) = sin(-x)
f(x) = -sinx
f(x) = 2sinx
f(x) = 0.5sinx
f(x) = sin(2x)

5. Trigonometric Identities

Trigonometric identities are equations where they are always true regardless of the value of θ.

5.1. Pythagorean Identities

sinθ =
opposite / hypotenuse
cosθ =
adjacent / hypotenuse
tanθ =
opposite / adjacent

When on a graph, we can use x, y, and r.

sinθ =
y / r
cosθ =
x / r
tanθ =
y / r

5.2. Reciprocal Identities

cscθ =
1 / sinθ
secθ =
1 / cosθ
cotθ =
1 / tanθ

5.3. Half-Angle Identities

sin(A+B) = sinAcosB + cosAsinB
sin(A-B) = sinAcosB - cosAsinB
cos(A+B) = cosAcosB - sinAsinB
cos(A-B) = cosAcosB + sinAsinB
tan(A+B) = (tanA+tanB)/(1- tanAtanB)

5.4. Double-Angle Identities

sin2θ = 2sinθcosθ
cos2θ = cos2θ - sin2θ or 2cos2θ-1 or 1-2cos2θ
tan2θ = 2tanθ / 1 - tan2θ
